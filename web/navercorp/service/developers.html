    • FOR DEVELOPERS

    네이버는 개발자와 개발사들을 위해
    앞선 지식과 경험을 공유하며 함께 성장해나갑니다.

    네이버랩스_
    생활환경지능 기반의
    기술 연구 기업

    2013년 네이버의 기술연구조직으로 시작한 네이버랩스는 2017년 1월 별도 법인으로 독립하여 자율주행, 로보틱스, 인공지능, 공간정보 등의 분야에서 생활환경지능(ambient intelligence)을 연구합니다. 네이버랩스의 비전인 생활환경지능은 사용자의 환경을 먼저 이해하고, 사용자가 요구하기 전에 필요한 정보와 서비스를 앞서 제공할 수 있는 기술을 의미합니다. 한국과 유럽의 우수한 연구진들이 공조하며 실제 사람들의 삶이 펼쳐지는 공간(location)에 대한 이해, 그리고 그 공간과 공간 사이를 연결해주는 이동(mobility)의 지능화에 집중하고 있습니다.

    네이버
    DEVELOPERS

    • 네이버 D2스타트업팩토리
      로보틱스, 모빌리티, 스마트홈 등 기술 분야의 스타트업을 발굴 및 육성하는 테크 스타트업 엑셀러레이터입니다.
    • 네이버 D2
      함께 성장하는 개발자 프로그램으로 가치있는 기술 지식 생산을 돕고, 경험을 나누며, 개발자를 지원합니다.
    • DEVIEW
      지식을 나누고 탁월함을 추구하며, 함께 성장하는 대한민국 대표 개발자 컨퍼런스입니다.
    • 네이버 오픈소스
      글로벌 기술 경쟁 체제 속에서 더 나은 경쟁력을 찾기 위해 네이버 오픈소스 기술을 함께 공유하고 나눕니다.

    네이버 API

    • 클로바 플랫폼 API
      클로바 플랫폼을 활용한 새로운 음성 기술과 서비스, 풍부한 콘텐츠를 누구나 쉽게 구축하고 사용할 수 있습니다.
    • 파파고 API
      다양한 국가의 언어에 대한 번역기능과 한국 인명을 로마자 표기법에 의한 표기로 변환하는 기능을 제공합니다.
    • 네이버지도 API
      웹 서비스나 모바일 앱에 네이버 지도를 삽입, 다양한 위치 기반 애플리케이션을 만들 수 있습니다.
    • 네이버 로그인
      별도의 아이디, 비밀번호 없이 네이버 아이디로 간편하게 외부 서비스에 로그인 할 수 있습니다.
